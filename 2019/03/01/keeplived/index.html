    keepalived and nginx 高可用负载均衡方案

    rpm包安装nginx和keepalived

    注意事项：离线安装需要解决依赖依赖问题，我是在内网redhat安装。
    1.更换redhat的yum源需要注册使用，更换centos免费yum源。yum地址修改为阿里云镜像地址，出现生产内网没有dns服务器不能解析依赖地址的问题。
    2.修改为方案二，通过一台公网服务器下载所需要的依赖包.
      yum install --downloadonly --downloaddir {dir} {soft}
    3.通过官网下载安装即可。
